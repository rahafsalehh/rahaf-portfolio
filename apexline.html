ApexLine (Duo-partnership)

A Formula 1-inspired platform for racing fans built using Python, Streamlit, and web technologies.

Partner: Arzoo Haroon 07437040325

Project Overview

ApexLine is a modern racing community website designed for F1 fans to view real-time stats, race insights, and driver performance. Built using Python, Streamlit, and JavaScript, the platform emphasizes interactive visuals and smooth user experience.

Technologies Used

Python
HTML5
CSS3
JavaScript
Streamlit

Key Features

  • Real-time race data and driver statistics
  • Responsive layout with mobile support
  • Interactive charts and race timelines using Streamlit
  • Clean UI inspired by modern F1 visuals
  • Fast backend logic written in Python
  • Lap simulator and driver filters to analyse fastest laps
