LandLordPro (Collaborative)

A cross-platform property management app connecting landlords and tenants with secure backend functionality.

Project Overview

LandLordPro is a complete property renting system that allows landlords to list properties, tenants to apply for rentals, and both parties to communicate through a secure platform. It features user authentication, payment tracking, and mobile-responsive design.

Technologies Used

GitHub
PHP
JavaScript
MySQL
HTML
CSS3

Key Features

  • Landlord dashboard to manage listings, tenants, and contracts
  • Tenant portal to browse and apply for rentals
  • Secure login system with role-based access
  • Payment tracking and rent due notifications
  • Database-driven backend with MySQL
