Cantor College

A institutional website dedicated to a fictional college based in Sheffield.

Project Overview

This project was my first ever created dynamic and responsive website built for Cantor College, aimed at serving both prospective and currently enrolled students. Developed using HTML, CSS, and JavaScript, the site provides essential information about the institution, academic programs, student services, and campus life.

Technologies Used

HTML5
CSS3
JavaScript
Github

Key Features

  • Fully responsive design across all devices
  • Responsive design for mobile and desktop devices
  • Clean, modern layout with interactive UI
  • Easy navigation to key resources for new and existing students
