SmartPay

A secure and efficient international payment system built with PHP, MySQL, HTML, CSS, and GitHub.

Project Overview

SmartPay is a user-focused currency transfer and payment management system. It includes secure login, responsive layout, and a dashboard for tracking all transaction history and balances — built with a full-stack PHP and MySQL backend.

Technologies Used

PHP
MySQL
HTML
CSS
GitHub

Key Features

  • User authentication and secure session handling
  • Custom dashboard with balance and transaction data
  • Responsive UI for all devices
  • Admin panel to manage users and payment flow
  • Database-driven design using MySQL
